# 配合Stylus美化Confluence的目录

大家在使用wiki的时候，会大量使用“目录”这个宏，但是内容一多想快速的跳转到指定目录就很麻烦，需要回到首页再去点击需要访问的目录链接。<br/>

## 步骤一
在Chrome扩展程序中安装Stylus插件，非chrome浏览器的扩展功能里若没有这个插件，只要这个浏览器有chrome内核，直接访问谷歌商店安装即可。<br/>

## 步骤三
